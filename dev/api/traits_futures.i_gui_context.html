Interface for GUI toolkit context objects.

class traits_futures.i_gui_context.IGuiContext[source]¶

Bases: abc.ABC

Interface for objects usable in a GUI context.

An instance of this class provides consistent mechanisms to get objects related to the event loop for a particular choice of GUI toolkit or event loop.
