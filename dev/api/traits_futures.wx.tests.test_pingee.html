gui_context_factory()[source]¶

IGuiContext implementation with lazily-determined toolkit.

The first time this context is used, an appropriate toolkit will be selected.

The toolkit selection mechanism used matches that used by Pyface, and is based on the value of the ETS_TOOLKIT environment variable, followed an examination of the available setuptools entry points under the entry point group “traits_futures.gui_contexts”.
